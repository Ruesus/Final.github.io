# Магазин футболок

## Опис
Простий інтернет-магазин футболок, створений для демонстрації базового функціоналу онлайн-магазину. Реалізований за допомогою HTML, TailwindCSS.

## Особливості
- Використання TailwindCSS для швидкого стилювання.
- Адаптивний дизайн для мобільних та десктопних пристроїв.
- Сторінки:
  - Головна сторінка з товарами
  - Сторінка товарів
  - Карточка з товаром
